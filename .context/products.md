# Products & Services

## UVRelining Products

### UV XAIR (Flagship)

Latest-generation LED UV curing system for pipe rehabilitation.

- Cures DN70 to DN600 with a single UV head (unmatched in market)
- Curing speeds up to 110m/hour via Flash Curing Process (FCP)
- Wireless control via Panasonic Toughbook G2 tablet
- Three-wheel puller unit with automatic anti-slip surveillance
- Cloud-based management panel with automatic quality report generation
- Lightweight carbon panel construction (racing industry-inspired)
- T-Liner System compatible (UV-cured T-liners and patch repairs)
- Hyperoptimized startup process with pre-impregnated liner

### UV360RS+

Compact UV light-head system. Slim body navigates 90-degree bends down to DN70. (Older model, being succeeded by UV XAIR)

### UVLCR (UV Light Curing Resin)

- Vinylester composition for maximum strength and durability
- Ultra-low VOC formulation
- Minimized curing time for fast installations

### UVHL (UV House Liner)

- High-performance house liner
- Available in multiple dimensions
- Market-leading reliability

## Researcher

Tool that simplifies complicated documents.

### How it works

1. User asks a question about a complex topic
2. System searches the web for relevant information
3. Explains findings in simple, step-by-step format
4. Provides diagrams and illustrations where helpful

### Document types

- ISO certifications
- Technical data sheets
- Norms and standards
- Regulatory documents
- Laws and legal applications
- Any complex documentation
